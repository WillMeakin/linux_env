Custom config for LogiOpts (https://github.com/PixlOne/logiops)
Used with MX MAster 3 and MX Keys
Place in /etc/logid.cfg

Frome an issue, "MX Keys for Mac" CIDs:
0xc7: monitor bright down
0xc8: monitor bright up
0xe0: window overview
0xe1: app overview
0xe2: kb bright down
0xe3: kb bright up
0xe4: media prev
0xe5: media play pause
0xe6: media next
0xe7: vol mute
0xe8: vol down
0xe9: vol up
0x0d: eject
0xd1: logi device 1
0xd2: logi device 2
0xd3: logi device 3
0x0a: calc
0xbf: screenshot
0xea: menu
0x6f: screen lock
0xec: arrow left
0xeb: arrow right			
0xde: (non-programmable)
0x34: (non-programmable)
